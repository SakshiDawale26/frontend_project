# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Babel](https://babeljs.io/) (or [oxc](https://oxc.rs) when used in [rolldown-vite](https://vite.dev/guide/rolldown)) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.

# frontend_project
A responsive frontend UI built using React.js and Vite, based on a Figma design. The project demonstrates clean component structure, CSS Modules, and responsive design.

## Objective
The objective of this project is to build a responsive and clean frontend user interface by converting a given Figma design into a React.js application. The focus is on implementing accurate UI layout, reusable components, and responsive design while following frontend best practices.

This project was developed as part of a frontend assessment to demonstrate practical skills in React.js and modern frontend development.

---

## Tech Stack Used
- Tech Stack (Mandatory)
- React.js (Vite or CRA)
- JavaScript (ES6+)
- HTML5 - CSS3 (CSS Modules / SCSS / Styled Components) 

---

## Features
- Fully responsive design (Desktop, Tablet, Mobile)
- Reusable and modular React components
- Pixel-aligned UI based on Figma design
- No external UI libraries used

  Figma Desing Link : https://www.figma.com/design/DsFXeaoygEhm80f0irb2p8/Assessment-Task
---
## Learnings from This Project
- Converting Figma designs into real React components
- Building reusable and maintainable UI components
- Implementing responsive layouts using CSS Grid and Flexbox
- Using CSS Modules for scoped and clean styling
- Structuring a React project following best practices
- Improving frontend problem-solving and debugging skills

---

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open `http://localhost:5173` in the browser

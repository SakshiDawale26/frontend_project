# frontend_project
A responsive frontend UI built using React.js and Vite, based on a Figma design. The project demonstrates clean component structure, CSS Modules, and responsive design.

## Objective
The objective of this project is to build a responsive and clean frontend user interface by converting a given Figma design into a React.js application. The focus is on implementing accurate UI layout, reusable components, and responsive design while following frontend best practices.

This project was developed as part of a frontend assessment to demonstrate practical skills in React.js and modern frontend development.

---

## Tech Stack Used
- Tech Stack (Mandatory)
- React.js (Vite or CRA)
- JavaScript (ES6+)
- HTML5 - CSS3 (CSS Modules / SCSS / Styled Components) 

---

## Features
- Fully responsive design (Desktop, Tablet, Mobile)
- Reusable and modular React components
- Pixel-aligned UI based on Figma design
- No external UI libraries used

## Learnings from This Project
- Converting Figma designs into real React components
- Building reusable and maintainable UI components
- Implementing responsive layouts using CSS Grid and Flexbox
- Using CSS Modules for scoped and clean styling
- Structuring a React project following best practices
- Improving frontend problem-solving and debugging skills
